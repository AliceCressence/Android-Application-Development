Student Grade Calculator
A native Android application built with Kotlin and Jetpack Compose for the SE 3242 Android Application Development course at ICT University. 
The app allows students to add subjects with their scores, and automatically computes grade letters (A+, A, B, C, D, F), pass/fail status,
overall average percentage, and GPA on a 4.0 scale. It is built using OOP with inheritance (Calculator base class → GradeCalculator derived class), 
Kotlin lambdas, null safety operators (?., ?:), and Material Design 3 components. To run the app, clone the repository, open it in Android Studio,
wait for Gradle sync to complete, then connect an Android device (API 26+) with USB Debugging enabled or launch an emulator, and click the green Run button
— the app will install and launch automatically.
